Enables the choice of payment options per product.

== Description ==

This plugin offers you the choice per product to select one or multiple payment gateways to use for this product.
When no options are checked, the default payment options will be shown, and if mutiple products are selected for purchase, the intersection of gateways for these products are shown.


== Installation ==

=Requirements=

-	Wordpress
-	Woocommerce

=Installation=

How to install this plugin:

1.	After download locate the “product-payment-options.zip” file on your computer
2.	Login into your Wordpress-site and go to Plugins > Add New to upload the new plugin.
3.	Click on Uploads in the top menu.
4.	Click Choose File and select the .zip file you had located in step 1. Press okay.
5.	Once you have your file selected, click on Install Now.
6.	Activate the plugin right after upload.
7.  Enjoy!
